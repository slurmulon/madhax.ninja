## Project setup
```
nvm use v20.31.1 // or higher
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Running Netlify API

```
npm install -g netlify-cli
netlify dev
```
